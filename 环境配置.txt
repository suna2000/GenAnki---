# genanki 运行环境配置

## Python 版本
- 最低版本: Python 3.8+
- 推荐版本: Python 3.11

## 依赖包
cached-property
frozendict
chevron
pyyaml
genanki (本地安装)

## Miniconda 安装命令

# 1. 创建环境
conda create -n env-anki python=3.11 -y

# 2. 激活环境
conda activate env-anki

# 3. 安装依赖 (推荐使用 pip)
pip install cached-property frozendict chevron pyyaml

# 4. 安装 genanki (本地开发模式)
cd C:\Users\25018359\Desktop\genanki-main
pip install -e .

## 运行脚本
python example.py
